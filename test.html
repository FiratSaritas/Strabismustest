Test-Einstellungen

Um den Test durchzuführen, benötigen Sie eine Brille mit zwei verschiedenen Farben, wie z.B. eine 3D-Brille.

Passen Sie die Farben unten an.

Anleitung zum Test
  1. Setzen Sie Ihre Rot/Cyan-Brille auf.
  2. Positionieren Sie sich etwa 30-50 cm vom Bildschirm entfernt.
  3. Halten Sie Ihren Kopf still und bewegen Sie den farbigen Punkt mit der Maus.
  4. Klicken Sie direkt auf die Mitte jedes der sechs roten Kreuze.
  5. Nachdem Sie auf alle sechs geklickt haben, nehmen Sie die Brille ab.
